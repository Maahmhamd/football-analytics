🎯 الميزات الرئيسية
═══════════════════════════════════════════════════════════════════════════

1. 📺 البث المباشر والإحصائيات الفورية
   • تابع المباريات المباشرة في الوقت الفعلي
   • إحصائيات محدثة لحظياً (امتلاك الكرة، الرميات، إلخ)
   • مؤشر بث حي ملون مع نبض متحرك

2. 🎥 الفيديوهات والتسجيلات
   • دعم تحميل وتشغيل الفيديوهات
   • للمباريات المسجلة والمعاد تشغيلها

3. 📊 جداول الترتيب المتقدمة
   • ترتيب الفرق حسب النقاط والفرق بين الأهداف
   • تحديث تلقائي بعد كل مباراة
   • عرض مفصل للإحصائيات

4. ⭐ أفضل الهدافين والإحصائيات الفردية
   • تصنيف الهدافين بالموسم
   • إحصائيات اللاعب (أهداف، تمريرات، ركلات جزاء، إلخ)
   • تقييم اللاعب ودقة الرمية

5. 👥 معلومات اللاعبين والفرق
   • قوائم لاعبي الفريق
   • بيانات شخصية (الطول، الوزن، الجنسية)
   • رقم القميص والمركز

6. 🔍 محرك بحث قوي
   • بحث فوري عن الفرق واللاعبين والبطولات
   • نتائج مقترحة من متعدد الكيانات
   • تصفية وتنقية النتائج

7. 📱 تصميم متجاوب
   • يعمل على جميع الأجهزة (هاتف ذكي، تابلت، حاسوب)
   • واجهة حديثة وسهلة الاستخدام
   • دعم اللغة العربية كاملاً

8. ⚡ أداء عالي وقوي
   • نظام cache من جانب العميل
   • معالجة سريعة للبيانات الضخمة
   • تحمل ضغط المستخدمين المتزامنين
   • فهارس قاعدة بيانات محسّنة

🚀 خطوات البدء السريعة
═══════════════════════════════════════════════════════════════════════════

1️⃣ التثبيت على XAMPP:

   ▸ انسخ جميع الملفات إلى: C:\xampp\htdocs\football-analytics\
   ▸ افتح phpMyAdmin: http://localhost/phpmyadmin
   ▸ أنشئ قاعدة بيانات باسم: football_analytics
   ▸ استورد ملف football_db.sql
   ▸ افتح: http://localhost/football-analytics/

2️⃣ التثبيت على Hostinger:

   ▸ حمّل الملفات عبر FTP إلى public_html/
   ▸ أنشئ قاعدة بيانات من لوحة التحكم
   ▸ استورد football_db.sql
   ▸ عدّل config.php ببيانات الاتصال
   ▸ افتح: https://yourdomain.com/football-analytics/

3️⃣ إضافة البيانات النموذجية (اختياري):

   ▸ استورد sample_data.sql لاختبار الموقع بسرعة
   ▸ تحتوي على 6 فرق + مباريات + لاعبين

📊 هيكل قاعدة البيانات
═══════════════════════════════════════════════════════════════════════════

الجداول الأساسية:

  📌 tournaments      → البطولات (الدوريات، الكؤوس، الدولية)
  📌 teams           → الفرق والنوادي
  📌 players         → اللاعبون (الاسم، المركز، الرقم)
  📌 matches         → المباريات (النتائج، الإحصائيات)
  📌 match_events    → أحداث المباراة (أهداف، بطاقات)
  📌 player_match_stats → إحصائيات اللاعب في كل مباراة
  📌 standings       → جدول الترتيب (النقاط، الفرق)
  📌 season_statistics → إحصائيات الموسم الكلية
  📌 users           → حسابات المستخدمين
  📌 notifications   → الإشعارات والتنبيهات

الفهارس المحسّنة:
  🔍 فهارس على حالات المباريات والتواريخ
  🔍 فهارس على الفرق واللاعبين
  🔍 فهارس على أنواع الأحداث
  🔍 فهارس مركبة للأداء الأفضل

🔌 واجهة API
═══════════════════════════════════════════════════════════════════════════

نقاط النهاية المتاحة:

1. المباريات:
   GET api.php?action=get_matches
   GET api.php?action=get_match_details&match_id=1
   GET api.php?action=get_live_matches

2. البطولات:
   GET api.php?action=get_tournaments
   GET api.php?action=get_tournament_standings&tournament_id=1

3. الفرق:
   GET api.php?action=get_teams
   GET api.php?action=get_team_details&team_id=1
   GET api.php?action=get_team_players&team_id=1

4. اللاعبون:
   GET api.php?action=get_top_scorers&limit=10
   GET api.php?action=get_player_stats&player_id=1

5. الإحصائيات:
   GET api.php?action=get_match_statistics&match_id=1

6. البحث:
   GET api.php?action=search&q=الكلمة

جميع الاستجابات بصيغة JSON مع دعم كامل للعربية.

      ✓ قلل عدد الاستعلامات

      ✓ استخدم روابط صور من مواقع خارجية موثوقة
      ✓ تحقق من السماح بالوصول للموارد
